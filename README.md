# Unstable Prototype

A 2D low horror game featured with exploration, secrets, and combat.



## Features Implemented

### System

- camera follow
- scene transition
    - scene loader
    - exit-spawn id match
    - gated exit
    - persistant items

### Character

- walk & flip
- jump & double jump
- dash
- swim & drown
- rope climb
- line renderer

### Environment

- tile map
- ground 
- water
- rope
- door
- locked door

### Scenes

- main menu
- remote village
    - house
    - garage
- tower front
- tower entry (working)

### Helpers

- bootstrap scene
- align and sync text
- asset stack (row and column)
